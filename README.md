# Inventory Management System - Ionic Angular

Mobile inventory management application built with Ionic and Angular.

## Project Structure

```
InventoryApp_Part3/
├── src/app/
│   ├── models/          # Data models and interfaces
│   ├── services/        # Business logic and API calls
│   ├── pages/           # Tab pages (List, Add, Update, Privacy)
│   ├── components/      # Reusable UI components
│   ├── guards/          # Route protection guards
│   └── tabs/            # Tab navigation
└── Documentation/       # Assessment documents
```

## Setup & Installation

1. Navigate to project directory:
   ```bash
   cd InventoryApp_Part3
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run development server:
   ```bash
   ionic serve
   ```

## Features

- **Tab 1**: List items with search functionality
- **Tab 2**: Add new items and view featured items
- **Tab 3**: Update and delete existing items
- **Tab 4**: Privacy and security settings
